## Overview
DSCN-G-BIO is the biological testability framework of the DSCN-G cognitive architecture. It bridges the gap between formal autopoietic graph dynamics and observable neurophysiological signals, operating strictly at Marr's (1982) algorithmic level (Computational Neural Correlates - NCC). 

This repository contains the theoretical paper, experimental protocols, and the Python analysis pipeline designed to test the framework's predictions using public EEG datasets and prospective laboratory collaborations.

### Core Contributions
1. **Neurobiological Mapping:** Functional-analogical mapping of DSCN-G elements to neural substrates (e.g., $E_i$ mapped to saliency signaling via LC-NE and mesocortical DA pathways; $\rho(t)$ as a computable proxy for integrated information $\Phi$).
2. **Prediction C3 (Phase-Hijacking):** A falsifiable, directional, and thresholded prediction regarding acute pain/salience processing, specifying unidirectional Granger causality (S1 → aPFC) in the 100-250 ms biological latency window.
3. **The Three-Phase Model of Deep Comprehension:** A formal dynamical model (Accumulation → Threshold → Sustained Flow) derived from contextual density $\rho(t)$, generating 8 distinct, resource-tiered experimental predictions (P1-P8).
4. **Methodological Honesty:** Explicit declaration of limitations, distinction between hypothesis-generating case studies (N=1) and required prospective protocols (ESM, N≥100), and rigorous control for EEG volume conduction artifacts.

## Analysis Pipeline (`src/dscn_bio_eeg_analyzer.py`)
The provided Python script is a self-contained testing environment that validates the mathematical predictions using synthetic EEG data before running on in-vivo datasets. 

**v4.0 Features:**
*   **Weighted Phase-Lag Index (wPLI):** Replaces standard PLV to mitigate volume conduction artifacts (ignores $0^\circ$ and $180^\circ$ phase differences from single-source skull propagation).
*   **Synthetic EMG Injection:** Injects broadband (30-150 Hz) exponential-decay noise at the `insight_point` to simulate facial micromovements (smiles, eye-opening) during the "Aha!" moment.
*   **Mock ICA Rejection:** Uses PCA-based component removal as a proxy for Independent Component Analysis to demonstrate pipeline robustness against motor artifacts.
*   **Granger Causality:** Implements Vector Autoregressive (VAR) modeling to test the unidirectional flow prediction (Sensory → Integrative regions).
*   **Contextual Density Proxy ($\rho(t)$):** Theoretically justified weighted combination of alpha (attentional window) and gamma (local integration) coherence.

## Installation & Usage
```bash
git clone https://github.com/Rylow999/dscn-g.git
cd dscn-g
pip install -r requirements.txt
python src/dscn_bio_eeg_analyzer.py
```
*Note: For production analysis on real human EEG data (e.g., MNE-Python pipelines), uncomment `mne>=1.6.0` in `requirements.txt` and replace the `mock_ica_rejection()` function with MNE's FastICA implementation.*
